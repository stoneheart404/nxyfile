-------------------------------------------------------
  WHAT IS CODIVEBOX
-------------------------------------------------------

CodiveBox is a desktop IDE built on Electron. It looks and
feels like VS Code but is fully self-contained. The core
pillars are:

  1. Code Editor     — CodeMirror 5 with syntax highlighting,
                       folding, autocomplete, split view.
  2. File System     — Virtual FS stored in localStorage,
                       with real folder open via Electron.
  3. Code Runner     — Python via Pyodide (in-browser WASM),
                       HTML preview via Blob URL.
  4. AI Chat         — Right-side panel, connects to any
                       OpenAI-compatible API (DeepSeek,
                       GPT-4o, Claude, Gemini).
  5. Shell/UI        — Custom titlebar, activity bar, sidebar
                       views (Explorer, Search, SCM, Debug,
                       Extensions, Settings), output panel,
                       status bar, command palette.

  KNOWN ISSUES (fix these first)
-------------------------------------------------------

  [ ] API key is stored in plain config.json — needs
      Electron safeStorage or keytar encryption
  [ ] Config is loaded from file but settings panel
      saves to localStorage only — they can desync
  [ ] "Open Folder" loads real files but doesn't sync
      back saves to disk (edits are lost on reload)
  [ ] SCM panel is purely decorative (hardcoded M files)
  [ ] Debug / Extensions / Search sidebar views are stubs
  [ ] Split editor copies active file content but the two
      panes don't stay in sync
  [ ] Pyodide loads fresh every cold start — no caching
  [ ] No error boundaries — a bad API key gives a raw
      fetch error in chat, not a friendly message
  [ ] Command palette arrow-key navigation is broken
      (only mouse click works)
  [ ] btnFix in toolbar has a malformed SVG path
