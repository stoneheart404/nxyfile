-------------------------------------------------------
  INSTRUCTIONS FOR AI ASSISTANTS
-------------------------------------------------------

When you install, update, upgrade, or change anything in
this project (dependencies, config, code structure, etc.),
you MUST append a new entry to the CHANGELOG section below.

Entry format:
  [YYYY-MM-DD HH:MM] | <type> | <short description>
  Details: <what changed and why>

  Types:
    - INSTALL   : New package or tool installed
    - UPDATE    : Existing package or dependency updated
    - UPGRADE   : Major version bump or structural change
    - CONFIG    : Configuration file changed
    - REFACTOR  : Code structure or architecture changed
    - BUGFIX    : Bug fix applied
    - FEATURE   : New feature added
    - REMOVE    : Package or file removed

Rules:
  1. ALWAYS append — never edit or delete existing entries.
  2. Use UTC time when possible.
  3. Include package versions when relevant (e.g., express 4.18 → 4.19).
  4. If multiple changes happen in one session, log each separately.
  5. Keep descriptions concise but specific.
